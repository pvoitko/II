# A Recurrent Neural Network (RNN) From Scratch

This was written for my [Introduction to Recurrent Neural Networks](https://victorzhou.com/blog/intro-to-rnns/).

## Usage

Install dependencies:

```bash
$ pip install -r requirements.txt
```

Run the RNN:

```bash
$ python main.py
```
